# CFT
Приложение, отображающее список пользователей бота и позволяющее администратору редактировать данные пользователя.

# В ролях
- Typescript
- Feathers + NeDB для прототипирования API
- RRR на клиенте
- и [многие-многие другие](package.json)

# Запуск
```shell
npm run start
npm run start:prod
```

Сборка
```shell
npm run build
npm run build:prod
```
